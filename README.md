# MONOLITH | Design Narrative

A high-fidelity digital portfolio built for the modern web. This project explores the intersection of structural architecture and fluid digital motion.

##  Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS v4 (Alpha/Beta)
- **Animation:** GSAP (ScrollTrigger) & Framer Motion
- **Smooth Scroll:** Lenis
- **Typefaces:** Syne & JetBrains Mono

##  Key Features
- **Editorial Layout:** Asymmetrical broken grid system.
- **Kinetic Typography:** Dynamic scale and fade effects on scroll.
- **Glassmorphism:** High-performance backdrop-blur navigation.
- **Structural UI:** Technical metadata and coordinate systems.
